# Simple Calculator with Streamlit

A simple web-based calculator built using Python and Streamlit. This application allows users to input mathematical expressions as strings and get the numerical result. It supports basic arithmetic operations and includes error handling for invalid expressions.

## App Link:
https://gemini-cli-calculator.streamlit.app/

## Features

*   **Basic Arithmetic:** Supports addition, subtraction, multiplication, and division.
*   **User-Friendly Interface:** Built with Streamlit for an intuitive web interface.
*   **Error Handling:** Provides clear messages for invalid mathematical expressions.

## Core Principles

This project adheres to the following core principles:

*   **Clean and Readable Code:** The code is structured in a modular way, with clear function and variable names, adhering to a consistent coding style.
*   **User-Friendly UI:** The user interface is simple and intuitive, with a single input field and a clear display of the result.
*   **Well-Commented Code:** Comments are added to explain complex logic, assumptions, and the purpose of different parts of the code.

## Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Mutahir-15/GIAIC_Q4_AIDD_30_Days_Challenge.git
    cd task_08/calculator
    ```

2.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

## Usage

To run the Streamlit application:

```bash
streamlit run src/main.py
```

This will open the application in your web browser.

## Running Tests

To run the unit tests:

```bash
python -m pytest
```
