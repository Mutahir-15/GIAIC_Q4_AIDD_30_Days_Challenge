# Quickstart

## Running the application

1.  Install the dependencies:
    ```bash
    pip install streamlit
    ```
2.  Run the application:
    ```bash
    streamlit run src/main.py
    ```

## Running the tests

1.  Install the dependencies:
    ```bash
    pip install pytest
    ```
2.  Run the tests:
    ```bash
    pytest
    ```
